Projects

Housing Prices Prediction using advanced regression techniques

Image Classification using Machine Learning Techniquues

Multi focus Image Fusion using Supervised Learning

Developed a global dictionary based on orthogonal matching pursuit algorithm combined with morphological filtering and sparse representation to fuse any two medical, multi-focus or sensory images together
